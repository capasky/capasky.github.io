Node Packages List

这里是一些nodejs的包的收集，便于检索及使用。

基础

Utils

  • lodash https://lodash.com
    lodash 是一个语言增强的库，可用于nodejs环境及浏览器环境。

Log

  • winston https://github.com/winstonjs/winston
    a multi-transport async logging library for node.js
    一个多路输出的异步的日志库。
  • log4js-node https://github.com/nomiddlename/log4js-node
    A port of log4js to node.js

工程化

  • gulp http://gulpjs.com/
    Automate and enhance your workflow
  • grunt http://gruntjs.com/
    Grunt: The JavaScript Task Runner

数据处理

DB Provider

  • node-mysql https://github.com/felixge/node-mysql
    A pure node.js JavaScript Client implementing the MySql protocol.
    MySQL的访问库。

ORM

  • node-orm2 https://github.com/dresende/node-orm2
    Object Relational Mapping
  • sequelize https://github.com/sequelize/sequelize
    Sequelize is an easy-to-use multi sql dialect ORM for Node.js & io.js. It currently supports MySQL, MariaDB, SQLite, PostgreSQL and MSSQL.
