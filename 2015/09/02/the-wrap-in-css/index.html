CSS中的Wrap

本文介绍以下几种与文字处理相关的CSS属性：

  1. white-space
  2. word-break
  3. word-wrap
  4. overflow-wrap

1. white-space

white-space CSS 属性是用来描述如何处理元素中的空格。
其初始值为normal。可取值及含义如下表：

换行符 空格和制表符 文字转行
normal 合并 合并 转行
nowrap 合并 合并 不转行
pre 保留 保留 不转行
pre-wrap 保留 保留 转行
pre-line 保留 合并 转行
  • wrap 表示对转行的处理，字面意思为包装，即是外部容器会包住内容，不侧漏，所以当容器中的内容遇到容器边缘时，会自动换行；
  • pre 是英文 preservation 的简写，即保留的意思，所以pre值会完全“忠于原著”；
    因此可以有以下理解：
    1. normal 默认值，该转行转行，该合并合并；
    2. nowrap 即是不进行自动换行处理；
    3. pre 保留所有的换行符、空格和制表符，不进行自动换行；
    4. pre-wrap = pre + wrap，为在pre的基础上，进行自动换行；
    5. pre-line = pre + line （only）？，为只保留换行符，其他按照normal进行处理。

2. word-break

word-break CSS 属性指定怎样在单词内断行。
其初始值为normal。可取值及意义：

  1. normal 使用默认的断行规则。
  2. break-all 对于non-CJK (中文/日文/韩文) 文本，可在任意字符间断行。
  3. keep-all CJK 文本不断行。 Non-CJK 文本表现同 normal。
